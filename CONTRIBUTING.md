# Contribuer

Merci de l'intérêt porté au projet. Quelques mots honnêtes pour cadrer les choses.

## L'état du projet

C'est un projet **personnel de recherche & développement**, mené en **solo** et très
expérimental. Rien n'est stable ni prouvé à grande échelle — l'inventaire honnête de
ce qui tient et de ce qui ne tient pas est dans le [README](README.md) et la
[revue de l'état du projet](docs/etat-du-projet.md).

## Le code

Le code est sous **Tous droits réservés** (voir [LICENSE](LICENSE)). Pour l'instant,
**je n'accepte pas de _pull requests_ de code** — pas par fermeture d'esprit, mais
parce que je garde le contrôle total tant que l'architecture bouge encore, et je ne
veux surtout pas que quelqu'un code un PR **pour rien**.

## Ce qui est vraiment bienvenu

À ce stade, les contributions les plus utiles ne sont pas du code :

- **Vos idées, retours et critiques** dans les [Discussions](../../discussions) —
  catégorie **Ideas** pour les suggestions, **Q&A** pour les questions.
- **Les signalements de bugs** via les [Issues](../../issues).

Ça compte beaucoup, et c'est ce qui fait le plus avancer un projet d'exploration.

## Soutenir le projet (optionnel, sans aucune contrepartie)

Je développe ça seul, sur mon temps. Si cette exploration vous parle et que vous
voulez en soutenir la suite, c'est possible et ça aide réellement — mais **sans
aucune promesse** : pas de fonctionnalité garantie, pas de calendrier, pas de
contrepartie. Juste un coup de pouce à de la R&D menée ouvertement et honnêtement.

> Le lien de soutien apparaîtra ici (et via le bouton « Sponsor » en haut du dépôt)
> une fois la plateforme choisie.
